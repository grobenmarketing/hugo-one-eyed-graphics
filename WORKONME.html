Technical Implementation Notes

Key Technologies

  • Framework: Hugo (Static Site Generator)
  • CSS: Custom CSS with CSS Grid and Flexbox
  • JavaScript: Vanilla JS for gallery filtering and masonry layout
  • Images: Netlify Image CDN for optimization (480w, 800w, 1200w)
  • Layout: CSS Grid Masonry with grid-auto-rows: 1px

Masonry Gallery Implementation

/* CSS Grid Masonry */ .gallery-grid { display: grid; grid-template-columns: repeat(auto-fill, minmax(300px, 1fr)); grid-auto-rows: 1px; gap: 1rem; } /* JavaScript calculates row spans based on image height */ /* item.style.gridRowEnd = 'span ' + rowSpan; */

Responsive Behavior

  • Desktop (1200px+): Full multi-column layouts, 3-column grids
  • Tablet (768px - 992px): 2-column grids, adjusted typography
  • Mobile (< 768px): Single column, stacked layout, burger menu, filter buttons stack vertically

Performance Optimizations

  • Lazy loading on gallery images (loading="lazy")
  • Async decoding (decoding="async")
  • Responsive images with srcset for bandwidth optimization
  • Debounced resize handlers (250ms)
  • Progressive masonry calculation per image
